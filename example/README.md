# Try commands

Execute _vc.lua_ without arguments to see usage.

See changes
```
./vc.lua 1.txt log
./vc.lua diff 2
./vc.lua diff -1
./vc.lua summ
```

Switch revision
```
./vc.lua rev 1
./vc.lua revm "Second"
./vc.lua rev
./vc.lua rev "new"
```

You can also save your changes (**add**), remove last commit (**pop**), merge a branch (**merge**), begin revision history from any saved versions (**base**), clear changes (**rm**).
